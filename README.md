# CodeAlpha_EDA

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Amazon Best Selling Books dataset. The analysis helps identify patterns, trends, distributions and relationships within the dataset.

## 📂 Dataset
Amazon Best Selling Books Dataset

## 🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## 📊 Features
- Data Cleaning
- Missing Value Analysis
- Statistical Summary
- Correlation Analysis
- Data Visualization
- Insights and Conclusions

## 📈 Visualizations
- Histogram
- Heatmap
- Countplot
- Boxplot

## 🎯 Objective
To explore and understand the structure of the Amazon Best Selling Books dataset and extract useful insights using EDA techniques.

## 📌 Conclusion
EDA helped identify important trends in book ratings, prices, categories and author popularity through statistical analysis and visualization.
